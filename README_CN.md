# 故事织机 — AI 关键词故事生成器

输入三个关键词，让 AI 为你编织一个故事。界面以梵高《星月夜》为灵感，采用油画质感背景、旋涡动画与洒落星光，营造梦幻的创作氛围。

## 功能特性

- **关键词生成** — 输入 3 个关键词，AI 将其编织成完整故事
- **风格与篇幅控制** — 支持温馨 / 暗黑 / 搞笑三种风格，短篇 / 中篇 / 长篇三种篇幅
- **流式输出** — 故事逐字呈现，实时查看 AI 创作过程
- **用户编辑区** — 可在生成后修改或续写故事内容
- **历史记录** — 侧边栏浏览、点击回顾、删除单条或清空全部；关闭页面时自动保存未保存的故事
- **多模型支持** — 支持智谱 GLM-4-Flash 与 DeepSeek V4 Flash
- **响应式设计** — 桌面端与手机端自适应，手机端侧边栏为抽屉式
- **梵高油画风格** — 油画渐变背景、旋涡装饰、暖金色调与洒落星光

## 开始使用

### 1. 打开应用

直接双击 `index.html` 用浏览器打开即可。

### 2. 配置 API Key

1. 点击右上角 **设置按钮**（⚙️）
2. 选择模型（GLM-4-Flash 或 DeepSeek V4 Flash）
3. 输入对应的 API Key，点击 **保存**

### 3. 生成故事

- 在输入框中输入 3 个关键词，或点击 **随机**（🎲）随机生成
- 选择 **风格** 和 **篇幅**
- 点击 **✨ 生成故事**（或按 Enter 键）

## API 配置

### 智谱 GLM-4-Flash

1. 在 [bigmodel.cn](https://open.bigmodel.cn/) 注册账号
2. 在控制台创建 API Key
3. 粘贴到应用的设置面板（GLM 区域）

### DeepSeek V4 Flash

1. 在 [platform.deepseek.com](https://platform.deepseek.com/) 注册账号
2. 创建 API Key
3. 粘贴到应用的设置面板（DeepSeek 区域）

> **注意：** API Key 存储在浏览器 `localStorage` 中，仅发送给对应的 AI 服务商。

## 技术细节

- **纯前端** — 无需构建工具或服务端代码
- **流式 API** — 使用 SSE/流式对话补全接口实现实时输出
- **数据持久化** — 历史记录与 API Key 通过 `localStorage` 存储；当前故事通过 `sessionStorage` 在刷新后恢复
- **响应式** — 使用 CSS 媒体查询，移动端断点 768px
